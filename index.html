What is this site?

This site was put together using Jekyll as the engine and hosted on a Github Pages. Special thanks to aksakalli for the template.

My reason for this site is to create maintainable documentation for some of the projects I have worked on.

I hope you find some useful information!

Easy to follow docs

The documentation provided by this site is thorough enough to follow as a tutorial, but can also be tweaked to meet a variety of development goals.

Working software
